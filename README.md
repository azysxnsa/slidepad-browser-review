## 主要功能拆解

**侧边吸附 + 自动呼出/隐藏**

Slidepad 打开后会自动吸附到屏幕的某一侧（可以是左右，也可以是角落）。鼠标靠近就滑出，离开就收起。也支持快捷键召唤，以及点击触发，灵活可选。

**任意网页都能添加**

这个很关键。它不像某些工具只能添加预设的几个 App，Slidepad 直接内置了一个地址栏，想加什么网站就加什么，Notion、飞书、Gmail、Todoist、各种 AI 工具，统统塞进去。

**窗口置顶，全屏也能用**

很多人喜欢全屏或者 Mission Control 分屏模式工作，Slidepad 的窗口会置顶显示，不会被全屏模式盖住。开着 Final Cut Pro 全屏剪视频，侧边一划，参考文件看完继续剪，这感觉真的比较爽。

**后台接收通知，隐藏也不断线**

把 Slidepad 收进侧边后，里面挂着的网站照样能收通知、播音乐。挂个网易云，一边工作一边听歌，Slidepad 缩回去了音乐还在放——这细节处理得很到位。

**可以检查页面元素**

这个功能让不少前端开发者挺惊喜：Slidepad 支持打开开发者工具检查页面元素。说明它内核是基于真实浏览器引擎做的，不是套壳网页框架。
